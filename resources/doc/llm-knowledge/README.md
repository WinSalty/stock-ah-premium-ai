# LLM 投资问答知识库

更新日期：2026-05-04

本目录专门存放面向智能问答的投资类知识材料。后端按用户问题和会话上下文选择性读取本目录下的 Markdown 与 DOCX 片段，避免每次问答载入全部项目文档，也避免把普通开发文档混入 LLM 参考材料。

## 分类

- `ah-premium/`：A/H 溢价、H/A 折价、港股通可操作性、跨市场替代、配对交易边界和 2026 AH 价差研究。
- `stock-selection/`：A 股蓝筹、低估值、红利、质量因子和估值分析框架。
- `financial-sector/`：银行、非银和 A 股长期资产筛选报告。
- `company-research/`：个股深度投资报告，目前包含五粮液 2026 投资报告。
- `macro-industry/`：宏观产业、地产金融、人口和资产负债表推演材料。
- `portfolio-risk/`：投资报告结构、风险提示、仓位和交易约束表达框架。

## 使用原则

- 文档只提供分析口径、术语解释、风险边界和报告模板。
- 精确数值以系统可用的市场数据为准，文档中的样例不作为实时事实。
- 面向用户回答时应使用专业投资分析语言，不暴露底层实现、SQL、视图名或数据落库方式。
- 新增给 LLM 使用的投研材料应先按主题放入本目录对应子文件夹，再在后端知识分类配置中登记关键词和文件路径。
