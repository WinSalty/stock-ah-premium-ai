# 投资问答知识分类

更新日期：2026-05-04

本目录存放面向智能问答的投资类知识材料。内容按分析主题拆分，供后端按用户问题和会话上下文选择性读取，避免每次问答载入全部项目文档。

## 分类

- `ah-premium-cross-market.md`：A/H 溢价、H/A 折价、港股通可操作性、跨市场替代与配对交易边界。
- `stock-selection-valuation.md`：A 股蓝筹、低估值、红利、质量因子和估值分析框架。
- `portfolio-risk-reporting.md`：投资报告结构、风险提示、仓位和交易约束表达框架。

## 使用原则

- 文档只提供分析口径、术语解释、风险边界和报告模板。
- 精确数值以系统可用的市场数据为准，文档中的样例不作为实时事实。
- 面向用户回答时应使用专业投资分析语言，不暴露底层实现、SQL、视图名或数据落库方式。
